# 像素艺术资源库

这里是一些高质量的开源像素艺术素材库，可以用来替换我们的角色精灵！

---

## 🎨 开源像素艺术素材库

### 1. **Universal-LPC-Spritesheet-Character-Generator** ⭐⭐⭐⭐⭐
- **地址**: https://github.com/Gaurav0/Universal-LPC-Spritesheet-Character-Generator
- **特点**: 可以通过组合身体部位、服装、装备来创建自定义角色
- **协议**: GPL 3.0 / CC BY-SA 3.0
- **优点**: 
  - 非常专业的像素风格
  - 可以组合不同的服装、发型、装备
  - 有完整的行走、跑步、攻击等动画
  - 统一的 LPC 标准

### 2. **OpenGameArt.org** ⭐⭐⭐⭐⭐
- **地址**: https://opengameart.org/
- **特点**: 大量免费的像素艺术游戏素材
- **推荐搜索**:
  - "RPG character sprites"
  - "pixel art character"
  - "visual novel character"
- **推荐**: "RPG character sprites" by Vircon32
  - 包含 2 个角色（冒险者、农民）
  - 有基本的行走、说话动画

### 3. **itch.io 免费像素艺术资源** ⭐⭐⭐⭐⭐
- **地址**: https://itch.io/game-assets/free/tag-pixel-art/tag-sprites
- **推荐项目**:
  - **Free Pixel Art Sprites and Animations** (KingBell)
    - 100+ 动画，300+ 物品
    - 适合几乎任何像素风格游戏
  - **Village NPC Pixel Character Sprites**
    - 可爱的乡村 NPC 像素角色包
  - **Pixel Art Character Creator Generator**
    - 在线生成角色的工具
  - **Mana Seed Character Base**
    - 免费的像素艺术 RPG 角色基础包

## 🎯 推荐方案

### 方案 1: 使用 Universal-LPC（推荐）
最专业、质量最高的选择：

```bash
# 克隆项目
git clone https://github.com/Gaurav0/Universal-LPC-Spritesheet-Character-Generator.git

# 启动本地服务器
cd Universal-LPC-Spritesheet-Character-Generator
python -m http.server

# 在浏览器访问 http://localhost:8000
```

然后我们可以：
1. 为银樱、狐狸先生、小萝、玩家分别设计角色
2. 下载生成的精灵图集
3. 集成到游戏中

### 方案 2: 在 itch.io 上找现成的角色
直接下载现成的像素角色包：
- 更快速、方便
- 有很多可爱的选择
- 可以直接使用

### 方案 3: 使用在线生成器
- https://kingbell.itch.io/pixel-sprite-mixer
- https://edermunizz.itch.io/pixel-art-rpg-character-creator
- 可以直接在浏览器生成角色，然后下载

## 💡 使用建议

### 角色风格建议
- **银樱**: 银色长发，蓝紫色连衣裙，温柔风格
- **狐狸先生**: 棕色头发/耳朵，棕色小西装，绅士风格
- **小萝**: 粉色头发，萝卜主题，可爱风格
- **玩家**: 便利店制服，中性风格
